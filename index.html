🚀 快速开始

1
克隆项目
git clone https://github.com/yourname/langchainjs-100-lines.git
2
安装依赖
cd langchainjs-100-lines && pnpm install
3
配置环境
cp .env.example .env
编辑 .env 文件，填入你的 API Key
4
运行示例
pnpm run dev:01
